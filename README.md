## Summary
A simple Python program that calculates a student's average grade, determines pass/fail status, and shows honors awards.

## Features
- Input student name and grades
- Calculates average grade
- Determines pass/fail status
- Assigns honors: Achiever, With Honors, With High Honors, Highest Honors

## How to Run
1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the file `gracal.py`.
4. Run the program in your Python environment.
5. Enter the requested information when prompted.
